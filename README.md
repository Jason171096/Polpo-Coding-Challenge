## Run Locally

Clone the project

```bash
  git clone git@github.com:Jason171096/polpo-Challenge.git
```

Go to the project directory

```bash
  cd polpo-Challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
